VShop Widget Plugin help you to integrate a VShop Banner

== Description ==
VShop Easy Widget Plugin help you to integrate a banner with products on your website to get money. Register on http://vshop.fr/ to get more informations.

== Installation ==
Set your VShop Key in your settings then use {vshop} or {vshop:keyword} in your article for get product banner.
